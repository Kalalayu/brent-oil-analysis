# Brent Oil Price Change Point Analysis

This project analyzes the impact of major geopolitical, economic, and policy events on Brent crude oil prices.  
Task 1 focuses on data preparation, exploratory data analysis (EDA), and compiling key oil market events to support later change point modeling and dashboard development.

## Task 1 Outputs
- Cleaned and prepared Brent oil price data  
- Initial EDA (trends, volatility, stationarity)  
- Structured event dataset  
- Interim report with findings and next steps  

## Tech Stack
Python, Pandas, Matplotlib, Statsmodels

## Next Steps
- **Task 2:** Bayesian change point modeling  
- **Task 3:** Interactive dashboard (Flask + React)
